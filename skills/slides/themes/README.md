### English

This skill generates standalone, interactive HTML presentations. It compiles predefined HTML templates, CSS themes, and JS logic into a single file based on user requirements.

**Features:**

- **Themes**: Cyberpunk, Corporate, Minimal, Nature.
- **Layouts**: Title, clean grids, code blocks, timelines, etc.
- **Responsive**: Optimized for both Desktop (slide view) and Mobile (scroll view).

### 中文

此技能用于生成单文件、交互式的 HTML 演示文稿。它根据用户需求，将预定义的 HTML 模板、CSS 主题和 JS 逻辑组合成一个文件。

**功能特点：**

- **多主题**：赛博朋克、商务、极简、自然。
- **多布局**：标题页、网格、代码块、时间轴等。
- **响应式**：针对桌面端（幻灯片视图）和移动端（滚动视图）进行了优化。
